# 🍎 Fruit Quality Detector

A deep learning-based web application that detects whether a fruit is **Fresh** or **Rotten** using image input.

---

## 🚀 Features

- Upload image or capture using camera
- Detect fruit quality (Fresh / Rotten)
- Confidence score visualization
- Clean and modern UI using Streamlit
- Real-time prediction

---

## 🧠 Model Details

- Architecture: MobileNetV2 (Transfer Learning)
- Classes: 10 (Fresh & Rotten variants)
- Validation Accuracy: ~99%
- Real-world Accuracy: ~90–95%

---

## 📸 How It Works

1. Choose input method (Upload / Camera)
2. Provide fruit image
3. Click Predict
4. Model analyzes image
5. Displays freshness result
